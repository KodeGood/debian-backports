Debian Backports APT Repo

Debian backports repository

To use:

sudo wget -O /usr/share/keyrings/kodegood-debian-backports.gpg https://kodegood.github.io/debian-backports/kodegood-debian-backports.gpg.key
sudo chmod 644 /usr/share/keyrings/kodegood-debian-backports.gpg
echo "deb [signed-by=/usr/share/keyrings/kodegood-debian-backports.gpg] https://kodegood.github.io/debian-backports any main" \
  | sudo tee /etc/apt/sources.list.d/kodegood-debian-backports.list

See the zhani/webruntime-apt-repo GitHub repository for more information.
